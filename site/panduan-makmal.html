UNTUK GURU TMK / JURUTEKNIK

Pasang Beramai-ramai di Makmal

Ada 30–100 komputer? Jangan pasang satu-satu. Pilih SATU daripada tiga cara di bawah — semuanya bermula dengan pemasang penuh yang anda jana di halaman Jana senarai.

⚠️ Jangan sesekali muat naik zip pemasang anda ke internet terbuka (laman web, pautan Drive "anyone with link") — ia mengandungi nama dan emel murid. USB dan rangkaian dalaman sekolah sahaja.
  1. A

    USB bergilir — tiada apa-apa prasyarat

    Ekstrak zip pemasang ke 2–3 pemacu USB. Semasa kelas, beri setiap murid beberapa komputer: cucuk USB, buka folder, klik dua kali pasang.bat — siap dalam ~10 saat sekomputer, terus dari USB tanpa salin dahulu. 100 komputer ≈ satu waktu kelas.

  2. B

    Folder kongsi rangkaian

    Jika makmal anda ada pelayan / folder kongsi: letak folder pemasang (sudah diekstrak) di \\PELAYAN\kongsi\quick-login\. Di setiap komputer, buka folder itu dan jalankan pasang.bat terus dari situ.

    Tahun baharu? Ganti folder di pelayan dengan pemasang baharu, jalankan semula pasang.bat di setiap komputer. Data murid kekal dalam rangkaian sekolah.

  3. C

    Veyon — satu klik untuk seluruh makmal

    Makmal yang sudah ada Veyon (perisian kawalan makmal — pemasangannya sekali sahaja oleh juruteknik) boleh menolak pemasangan ke semua komputer serentak. Gabungan terbaik: Veyon + folder kongsi (Pilihan B).

    Dalam Veyon Master, pilih semua komputer → Run program → taip: 

    cmd /c \\PELAYAN\kongsi\quick-login\pasang.bat <NUL

    (<NUL melangkau tetingkap "tekan apa-apa kekunci" supaya pemasangan selesai tanpa sentuhan. Veyon biasanya menjalankan program dalam sesi pengguna yang sedang log masuk — pemasangan itu sendiri tidak perlukan hak admin. Jika ikon tidak muncul di Desktop murid, semak tetapan sesi "Run program" Veyon anda.)

    Tiada folder kongsi? Anda boleh tolak muat-turun pemasang contoh (tanpa senarai sebenar) ke Desktop setiap komputer, kemudian seorang murid klik pasang.bat di setiap satu: 

    powershell -NoProfile -Command "iwr 'https://quicklogin.davinhub.com/muat-turun/quick-login-delima.zip' -OutFile $env:TEMP\ql.zip; Expand-Archive $env:TEMP\ql.zip ([Environment]::GetFolderPath('Desktop')+'\quick-login') -Force; Remove-Item $env:TEMP\ql.zip"

    Senarai murid sebenar kemudiannya perlu sampai melalui USB atau folder kongsi — bukan internet.
